RPG Pixel Character Generator

Tagline: Pixel Heroes: Your RPG, Your Characters!

Overview

Create fully animated RPG pixel characters in seconds. Customize body, hair, outfits, accessories, and weapons, with palette swaps and randomization. Export ready-to-use sprite sheets for Unity, Godot, or any engine.

Features

Character Engine:

Layer compositing: Body / Hair / Outfit / Accessory / Weapon.

Animation frames: Idle, Walk, Attack (4 frames each).

Recolor engine: Palette swapping & hue shifts.

Randomize presets and "reroll" with seed support.

Save/load character JSON configuration.


Export:

Export individual sprite sheets per animation: char_idle_4f.png, char_walk_4f.png, char_attack_4f.png

Combined sprite sheet: char_all_12f.png

Client-side ZIP packaging via JSZip: character_pack.zip



Installation & Usage

1. Open index.html in a modern browser.


2. Select character parts, recolor palettes, and preview animations.


3. Export sheets or ZIP for use in your project.


4. Save character JSON for future edits or rerolls.
